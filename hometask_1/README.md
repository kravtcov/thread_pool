The first home task: Set
===
Class Set implementation

Building MySet
---
In source folder, run:

    mkdir workspace
    cd workspace

for linux:

    cmake ../
    make

for MinGW:

    cmake -G "MinGW Makefiles"  -D CMAKE_MAKE_PROGRAM=mingw32-make.exe ../
    mingw-32-make

Running the programm:
---
    ./my_set
